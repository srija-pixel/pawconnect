# 🐾 PawConnect

A **full-stack Pet Care, Adoption, and E-commerce Web Application** designed to provide pet lovers with a seamless platform for adoption, consultation, nutrition guidance, and shopping.

---

## 🚀 Features

### 🏠 Homepage

* Modern UI with responsive design
* Navigation across all modules
* Canvas animation (dog chasing a ball)

---

### 🐶 Breeds Information

* Dog and Cat breed tables
* Audio (bark/meow) and video support
* Dynamic breed search
* Displays:

  * Breed details
  * Temperament
  * Media content

---

### 🍲 Pet Food & Nutrition

* General food habits for pets
* Dynamic nutrition recommendation system
* Supports **any breed input**
* Intelligent fallback logic if API fails

---

### 🧸 Accessories Store

* Product catalog:

  * Collar, Bed, Bowl, Toy, Leash
* Add to cart functionality
* Remove items from cart
* Dynamic total calculation
* Place order feature

---

### 🩺 Vet Consultation

* Select veterinarian
* Auto-fill doctor selection
* Submit consultation request
* 📧 Email confirmation sent using Nodemailer

---

### 📍 Nearby Pet Markets

* Uses Geolocation API
* Displays current location
* Embedded Google Maps
* Shows nearby pet shops and available breeds

---

### 🐾 Adoption System

* Browse pets available for adoption
* Modal-based adoption form
* Adoption policy popup
* Form validation
* 📧 Email confirmation on submission
* Status updates:

  * Under Review
  * Approved

---

## 🛠️ Tech Stack

### Frontend:

* HTML5
* CSS3 (Flexbox, animations, responsive design)
* JavaScript (DOM, Fetch API, Async/Await)

### Backend:

* Node.js
* Express.js
* Nodemailer (Email service)

## 📌 Key Highlights

* Full-stack working project
* Real-time email notifications
* Dynamic UI with modern design
* API integration with fallback logic
* Canvas animation implementation
* E-commerce cart system
